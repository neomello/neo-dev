# Como Publicar no Scalar Docs

## Passo a Passo

### 1. No Dashboard Scalar

1. Vá em **Docs** (menu lateral)
2. Clique em **Create Documentation** ou edite a existente (`@neomello/neo-dev`)

### 2. Configurar Source

**Opção A: GitHub Repository**
- Repository: `neomello/neo-dev`
- Branch: `main`
- Path: `projects/playground/openapi.yaml`

**Opção B: URL Raw (mais simples)**
- URL: `https://raw.githubusercontent.com/neomello/neo-dev/main/projects/playground/openapi.yaml`

### 3. Publicar

1. Clique em **Publish** ou **Save**
2. A documentação ficará acessível publicamente

## Verificação

Após publicar, você poderá:
- Ver o layout da documentação
- Validar clareza visual
- Compartilhar o link (se quiser)

## Nota

Esta é apenas para **ver o layout**. Não é para produção ainda.
